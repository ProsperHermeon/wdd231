Course Purpose

This course strengthens skills in semantic HTML, responsive CSS, accessible design, and modern JavaScript so you can plan and build quality web experiences for real users.

Outcomes

Apply best practices for layout, usability, and performance while collaborating effectively and integrating data from services and APIs.

Focus

Hands-on projects include a chamber of commerce site and a cumulative final that demonstrate responsive design and client-side interactivity.

Learning Activities

Each week blends readings, coding practice, group collaboration, and audits to keep skills measurable and repeatable.

  • Semantic markup and meaningful structure
  • CSS layout with flexbox, grid, and media queries
  • JavaScript modules, events, and asynchronous data
  • Accessibility, Lighthouse, and performance budgeting
